= Easing Functions

== Introduction

video::discontinuity.mp4[align="center"]

image::discontinuity.png[align="center"]

Easing functions are functions used to create <</reference/animations/basics/explanations#,smoother animations>>.
We say a function stem:[f] is an easing function if

[stem]
++++
f(0) = 0 \qquad f(1) = 1
++++

The simplest easing function is the linear one:

[stem]
++++
f(t) = t
++++

which goes from stem:[0] to stem:[1] in a straight line:

image::linear.png[align="center"]
