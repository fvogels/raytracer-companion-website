= Stratified Sampler

== Definition

A stratified sampler takes two parameter stem:[N] and stem:[M]. Given a rectangle to be sampled,
it subdivides this rectangle into stem:[N \times M] subrectangles of equal size.
The sampler then returns the center point of each of these rectangles.
