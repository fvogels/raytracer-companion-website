= Spiral Pattern

[overview]
----
difficulty 3
requires patterns/lambda
----

The goal of this extension is to replicate these results (a spiral pattern) without help.
You can try to solve this extension partially of partial rewards.

== Spiral Thickness

The thickness of the spiraling arm must be settable.

[source,language='chai']
----
var t = Animations.animate(0.1, 0.4, seconds(3))[now]
spiral([ "thickness": t ])
----

video::thickness.mp4[align="center"]

== Arm Count

We should be able to choose the number of arms.

[source,language='chai']
----
spiral([ "thickness": 0.1, "arms": arms ])
----

== Spacing

The spacing between arms must be a parameter too.

[source,language='chai']
----
spiral([ "thickness": 0.1, "spacing": spacing ])
----
